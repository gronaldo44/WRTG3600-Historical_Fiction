LLMs and Biological Computing

2000

Using web crawling technology, researchers at Max Planck Institute begin work on making the multi-brain machine, Us, crawl the internet to understand and use its collective information.

2006

Cloud computing has become commercially available. Max Planck Institute begins work on a project to use the multi-brain machine, Us, as the backend for an intelligence assistant.

2007

The project evolves into a smartphone application that takes in voice or text input and outputs a response from the machine. The technology is dubbed a large language model (LLM) and is only possible via the thinking patterns of a brain.

2010

Us launches as a standalone open source application on the world wide web and mobile devices. It is quickly adopted as part of the workflow for all thinking jobs. Schoolwork, secretary work, creative writing, and engineering work are just some of the fields that begin to see transformations.
